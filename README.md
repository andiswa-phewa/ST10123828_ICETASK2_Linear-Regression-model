# Medical Insurance Cost Prediction

This project explores a medical insurance dataset and builds a Linear Regression model to predict insurance charges.

## Dataset
The dataset contains information about individuals including:
- age
- sex
- bmi
- number of children
- smoking status
- region
- insurance charges

## Project Steps
The notebook includes the following steps:

1. Importing required libraries
2. Loading and exploring the dataset
3. Exploratory Data Analysis (EDA)
4. Data preprocessing
5. Feature and target selection
6. Train-test split
7. Training a Linear Regression model
8. Making predictions
9. Evaluating model performance
10. Reflection on results

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook
